# UCL Guessr

🎉 **Celebrating UCL's 200th Anniversary (1826-2026)** 🎉

## About the Game

UCL Guessr is an interactive campus trivia game created to celebrate University College London's bicentennial anniversary. Designed specifically for first-year students, this engaging web application helps newcomers navigate and discover the rich history of UCL's iconic campus buildings.

### 🎯 Game Features
- **Interactive Campus Map**: Explore UCL's historic buildings with an interactive Folium map
- **Timed Trivia Challenge**: Answer questions about 10 key campus locations within 90 seconds
- **Visual Learning**: View high-quality images of each building alongside trivia questions
- **Progress Tracking**: Real-time scoring and progress monitoring
- **Immersive Experience**: Fullscreen splash screen and professional game interface

### 🏛️ Why UCL Guessr?
As UCL marks its 200th year of pioneering education and research, this game serves as a digital guide for new students to:
- Learn about UCL's most important buildings and their historical significance
- Navigate campus with confidence from day one
- Discover fascinating facts about their new academic home
- Connect with UCL's proud heritage and traditions

Perfect for orientation week, campus tours, or independent exploration, UCL Guessr transforms campus familiarization into an exciting, interactive experience that celebrates two centuries of UCL excellence.

## Setting Up a Virtual Environment

1. **Create a virtual environment**:
    ```sh
    python3 -m venv venv
    ```

2. **Activate the virtual environment**:
    - On macOS and Linux:
        ```sh
        source venv/bin/activate
        ```
    - On Windows:
        ```sh
        venv\Scripts\activate
        ```

## Installing the Project
To install the project, use the following command:

```bash
pip install -e .
```

## Installing  Dependencies
To install dependencies listed in the requirements.txt file, use the following 
command:

```bash
pip install -r requirements.txt
```

## Running the Application
To run the UCL Guessr app locally, use the following command:

```bash
streamlit run frontend.py
```

The application will start on your local server and automatically open in your default web browser.
